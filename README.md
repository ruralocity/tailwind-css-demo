# Tailwind CSS demo

This is a basic starting point to demonstrate Tailwind CSS at Astoria
Full Stack, October 2020.

## Setup

1. Clone this repo
2. `npm install`
3. `npm install -g live-server`
4. `live-server public`


## Build/rebuild CSS

1. `npm run build`
